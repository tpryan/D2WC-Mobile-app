Skip the IDE with PhoneGap Build

Terrence Ryan

PhoneGap is a fantastic idea: build mobile applications using your pre-existing HTML/JS/CSS skills. However one of the downsides of cross platform mobile development remains - having to install and maintain multiple IDEs. PhoneGap Build is the answer. PhoneGap Build allows you from Raw HTML to mobile app in moments. This session will take you through that, and show you how you can use PhoneGap Build to build mobile apps faster today, for free.
